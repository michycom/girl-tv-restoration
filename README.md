# girl.tv Reborn: Restoring a Pioneering Net-Art Ecosystem

## 🚨 Project Status: Legacy Restoration
This repository contains the historical archive and legacy code of **girl.tv**, a pioneering social internet art project from Hamburg (active since the late 90s/early 2000s). The project was exhibited at the **Kunsthaus Graz** and connected ~50 artists in a pre-Facebook decentralized network.

Currently, the codebase exists primarily as **documentation (PDFs)** and fragmented legacy scripts that are no longer executable on modern systems. This project aims to **extract, restore, and migrate** this "broken" ecosystem into a functional, open-source platform using AI assistance.

## 🎯 Goal: Migration to the AI Era
The objective is not just preservation, but **reactivation**:
1.  **Extraction:** Using LLMs (Claude Max) to parse code and logic from static PDF documentation.
2.  **Refactoring:** Translating obsolete code (Perl, old HTML/CGI) into modern, secure standards.
3.  **Revival:** Re-launching girl.tv as an open-source blueprint for **artist-run, non-algorithmic social networks**.

## 🌍 Ecosystem Impact
This project represents a **"silent dependency"** of digital culture:
*   **Historical Value:** It is one of the few surviving prototypes of early, non-commercial social networking.
*   **Educational Resource:** The restored code will serve as a reference for artists and developers interested in **Net Art history** and decentralized web structures.
*   **Infrastructure:** Without restoration, this unique layer of internet history risks being lost forever.

## 🛠️ Technical Challenge
The primary hurdle is the **format of the archive**: critical logic is trapped in unstructured PDFs and outdated scripts. Advanced AI context windows are required to:
*   Reconstruct file structures from documentation.
*   Identify and fix "broken" dependencies.
*   Generate a migration path to modern frameworks.
