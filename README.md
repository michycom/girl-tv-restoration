# Hamburg Cultural Heritage Restoration Project

## 🚨 Project Status: Legacy Restoration & AI Migration
This repository serves as the central technical hub for restoring **multiple critical cultural archives** led by artist and curator **Michael Heering** (Drostei Prize 2017, former Director of Galerie Genscher).

The projects included are currently trapped in **fragmented legacy formats** (PDF documentation, obsolete code, decaying video tapes) that are inaccessible to modern systems. This initiative aims to **extract, restore, and migrate** these "broken" ecosystems into functional, open-source platforms using AI assistance (Claude Max).

### 🏛️ Included Archives
1.  **girl.tv:** A pioneering **Net-Art social network** from Hamburg (active late 90s/early 2000s). Connected ~50 artists in a decentralized, pre-Facebook environment. Exhibited at **Kunsthaus Graz** (Austria).
2.  **Richie Löffler Archive:** Digitalizing the legacy of **Richie Löffler**, one of Germany's first pro skateboarders (4th place World Championships 1996). Includes rare skate videos, shop history (Mantis), and subculture documentation.
3.  **Galerie Genscher & Others:** Preserving documentation from the non-profit gallery era (2007–2019) and various collaborative art projects in Hamburg's Karolinenviertel.

## 🎯 Goal: A Unified Restoration Pipeline
The objective is not just preservation, but **reactivation**. We are building a reusable **Open Source Restoration Pipeline** to save endangered cultural data:

1.  **Extraction:** Using LLMs (Claude Max) to parse code, logic, and metadata from static PDFs and unstructured documents.
2.  **Refactoring:** Translating obsolete code (Perl, old HTML/CGI) and analog data into modern, secure, and interactive web standards.
3.  **Revival:** Re-launching these projects as **open-source blueprints** for artist-run, non-algorithmic social networks and cultural archives.

## 🌍 Ecosystem Impact: The "Silent Dependency"
These projects represent a **"silent dependency"** of digital and subcultural history:
*   **Historical Value:** `girl.tv` is one of the few surviving prototypes of early, non-commercial social networking. The Richie Löffler archive documents a pivotal era in European skate culture.
*   **Educational Resource:** The restored code and data will serve as a reference for artists, developers, and historians interested in **Net Art**, **Skate History**, and decentralized web structures.
*   **Infrastructure:** Without AI-assisted restoration, this unique layer of cultural history risks being lost forever due to technical obsolescence. The tools developed here will be applicable to *hundreds* of similar endangered archives.

## 🛠️ Technical Challenge
The primary hurdle is the **format of the archives**: critical logic and content are trapped in:
*   Unstructured PDF documentation (code snippets, diagrams).
*   Obsolete server scripts (Perl, CGI) that no longer run on modern systems.
*   Analog video tapes and fragmented digital files.

**Why Claude Max?**
Advanced AI context windows are required to:
*   Reconstruct file structures from documentation.
*   Identify and fix "broken" dependencies across thousands of lines of legacy code.
*   Generate a migration path to modern frameworks (React, Node.js, etc.) while preserving the original "spirit" and logic of the early internet projects.
